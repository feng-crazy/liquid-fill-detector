# Optimization plan
针对使用传统机器学习算法在受阴影和反光干扰下检测缺陷的问题，这确实是一个经典的机器视觉难题。由于处理后的二值图显示存在阴影遮挡和液滴反光干扰，直接通过简单的全局阈值分割很难得到完美的缺陷判断。

为了解决“如何准确识别缺陷点”以及“如何设定面积阈值”的问题，建议采用以下基于传统图像处理的流程。这套方法不依赖深度学习，而是利用图像的几何特征和局部统计特性。

### 1. 图像预处理（去噪与增强）

由于存在反光（通常表现为高亮噪点）和阴影（表现为低亮度区域），直接二值化会丢失信息。

- **形态学开运算：** 使用一个较小的结构元素（如圆形或方形，半径3-5像素）对原图进行开运算。这有助于去除液滴反光产生的细小白点，同时保留较大的试管口区域。
- **自适应阈值：** 不要使用全局固定阈值。由于存在阴影，图像不同区域的亮度不一致。应使用**自适应阈值**（如`cv2.ADAPTIVE_THRESH_GAUSSIAN_C`），它根据图像局部邻域计算阈值，能有效应对光照不均。

### 2. 定位网格结构（ROI提取）

为了准确判断哪个位置缺失，首先要建立“标准坐标系”。

- **霍夫变换或轮廓检测：** 检测图像中的直线或主要轮廓，确定整个8x12阵列的边界框。
- **网格划分：** 根据阵列的几何规律（8行12列），通过透视变换将倾斜的阵列校正为正视图，或者直接根据间距计算每个试管口的理论中心坐标$(x_i, y_i)$。
- **建立感兴趣区域：** 为每一个理论坐标建立一个小的感兴趣区域窗口（例如，以理论中心为圆心，半径为$r$的圆形区域）。

### 3. 特征提取与缺陷判定（核心逻辑）

这是解决“面积阈值如何判断”的关键步骤。与其设定一个固定的“面积阈值”（例如“小于100像素就是缺陷”），不如采用**相对特征**或**统计特征**。

#### 方法一：基于局部对比度的判定（推荐）

不要只看面积，要看“填充率”或“平均灰度”。

- **填充率计算：** 在每个感兴趣区域内，计算白色像素（前景）占总像素的比例。
    - 正常试管口：填充率较高（例如 > 0.6）。
    - 缺陷（缺失）：填充率极低（例如 < 0.1）。
    - 反光干扰：虽然面积可能小，但通常位于边缘，或者通过形态学处理后可以区分。
- **动态阈值设定：** 计算整幅图像所有感兴趣区域的平均填充率 $\mu$ 和标准差 $\sigma$。设定动态阈值 $T = \mu - k\sigma$（$k$通常取2或3）。如果某区域的填充率低于 $T$，则判定为缺陷。这种方法可以自动适应不同的光照条件。

#### 方法二：基于几何形状的判定

利用“圆度”或“凸包”特征。

- **圆度：** 试管口通常是圆形的。计算每个连通域的圆度 $C = \frac{4\pi \times Area}{Perimeter^2}$。正常的试管口圆度接近1。反光噪点通常形状不规则，圆度较差；阴影可能导致形状畸变。
- **最小外接圆误差：** 对每个检测到的白色区域做最小外接圆，计算区域面积与外接圆面积的比值。

### 4. 解决“面积阈值”的具体策略

针对您提到的“面积过小”难以设定阈值的问题，可以采用以下策略：

- **直方图双峰法：** 统计所有检测到的连通域面积，绘制直方图。通常会出现两个峰值：一个峰值代表“背景/噪点”（面积小），另一个峰值代表“正常试管口”（面积大）。阈值应设定在两个峰值之间的谷底。
- **基于预期尺寸的归一化：**
    - 在理想无缺陷情况下，试管口的直径是已知的（或者可以通过图像中最大、最完整的几个点估算出来）。
    - 设理论最大面积为 $A_{max}$。
    - 设定阈值为 $\alpha \times A_{max}$，其中 $\alpha$ 是一个经验系数（例如0.4或0.5）。如果检测到的面积 $A_{current} < \alpha A_{max}$，则视为缺陷。

### 5. 总结流程图

1.  **输入图像** -> 2. **高斯模糊/中值滤波**（去反光噪点） -> 3. **自适应阈值二值化**（去阴影影响） -> 4. **形态学闭运算**（连接断裂的边缘） -> 5. **轮廓查找** -> 6. **网格匹配**（将轮廓归属到8x12的格子里） -> 7. **特征计算**（计算每个格子里轮廓的面积/圆度/填充率） -> 8. **动态阈值比较** -> 9. **输出缺陷坐标**。

### 针对反光干扰的特别提示

第一张图中，试管尖端液滴的反光非常亮。可以在二值化之前，先检测图像中的**极高亮区域**（例如像素值 > 240），将这些区域标记为“不可靠区域”或直接进行修补（Inpainting），防止它们被误判为试管口的一部分，或者干扰面积计算。

通过上述方法，特别是**动态阈值**和**网格化局部分析**，可以有效避开固定面积阈值难以设定的问题，提高在复杂光照下的检测准确率。
